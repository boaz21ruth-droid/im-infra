群聊增强 — 第二批功能

三个独立模块，逐一确认设计范围

👑

① 管理员权限 UI

群主在"群管理"页设置/撤销管理员；管理员可禁言成员；成员列表显示角色徽章

纯 Flutter 前端 · 调用 OpenIM SDK setGroupMemberRole / muteGroupMember

📁

② 群文件库

群成员上传文件；群设置页入口"群文件"；文件列表 + 下载；上传走自定义消息 type 914

Flutter 前端 + 历史消息检索 · 无需后端改动

🗳️

③ 投票

发起人在输入框菜单创建投票（标题 + 选项）；chat 气泡展示选项 + 实时票数；每人限投一次

自定义消息 type 909（poll）· 投票结果通过自定义消息回传更新

点击选中你想深入讨论的功能，或直接在终端告知
